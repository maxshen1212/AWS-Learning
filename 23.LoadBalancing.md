# VPC 網路架構

AWS Virtual Private Cloud (ex. 172.16.0.0/16)
└── Public Subnet (ex. 172.16.10.0/24)
-----└── WEB-EC2 (私有 IP: 172.16.10.10/32)
-----└── 綁定 Elastic IP（公網 IP）
└── Public Subnet (ex. 172.16.10.0/24)
-----└── WEB-EC2 (私有 IP: 172.16.11.10/32) -負載均衡設置
-----└── 綁定 Elastic IP（公網 IP）
└── Private Subnet (ex. 172.16.20.0/24)
-----└── DB-EC2（ex. 172.16.20.10/32）
-----└── 通常不直接對外連線（無公網 IP）
